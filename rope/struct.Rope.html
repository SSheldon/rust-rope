A Rope is a tree of Strings that allows more efficient storage and manipulation of large amounts of text than a String.

fn split(self, index: uint) -> (Rope, Rope)

Splits the Rope into two Ropes at the given index. Returns the pair of Ropes to the left and right of the split.

Failure

If index is greater than the length of the Rope. If index is not a valid character boundary.

Example

let rope = Rope::from_string("abcd".to_string());
let (left, right) = rope.split(2);
assert!(left.equiv(&"ab"));
